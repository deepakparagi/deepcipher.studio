## Ⅴ. Project Architecture

The repository is structured for modularity, scale, and maintainability.

```text
deepcipher-agency/
├── src/
│   ├── app/                # Next.js App Router (Pages & API Routes)

│   └── styles/             # Global CSS & typography tokens
```

## Ⅷ. Development Setup

To run the DEEPCIPHER environment locally, ensure you have Node.js 18+ installed.

```bash
# Clone the repository
git clone https://github.com/deepakparagi/Deepcipher.git

# Navigate into the project
cd Deepcipher_agency

# Install dependencies
npm install

# Start the development server
npm run dev
```
The application will be available at `http://localhost:3000`.

To create a production build:
```bash
npm run build
npm run start
```

## Ⅸ. Deployment

This platform is engineered for Vercel's Edge Network.

1. Push your code to your GitHub repository.
2. Import the project into Vercel.
3. Ensure the Build Command is set to `next build` and Output Directory to `.next`.
4. Add your environment variables (e.g., SMTP credentials for forms).
5. Deploy.
