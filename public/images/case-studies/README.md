Place your project screenshot images in this directory. 
You can create subfolders for each project, for example:
- `/public/images/case-studies/project-name-1/`
- `/public/images/case-studies/project-name-2/`

Then you can reference them in your case-study code like:
`<img src="/images/case-studies/project-name-1/screenshot1.png" alt="Screenshot" />`
